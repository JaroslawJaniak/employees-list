## Run shorthand project

1. npm install
2. npm run mock-server
3. npm run dev

## Run Locally

Clone the project

```bash
  git clone https://github.com/JaroslawJaniak/employees-list.git
```

Go to the project directory

```bash
  cd employees-list
```

Install dependencies

```bash
  npm install
```

Start json-server

```bash
  npm run mock-server
```

Start the server

```bash
  npm run dev
```
